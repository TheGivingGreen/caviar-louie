# Caviar Louie

St. Louis caviar bar concept. Mobile-first brand. Tableside service, grower champagne, celebration.

This repo holds the single-page marketing site.

## Files

- `index.html`. The site. Single HTML file, embedded CSS and JS.
- `Caviar_Louie.png`. The crest logo (transparent background).
- `hero.mp4`. The hero background video. Muted, looped, autoplay.
- `CAVIAR-LOUIE-DESIGN.md`. Full design system. Color tokens, typography, components, section architecture. The AI agent reference for any future builds or revisions.

## View it

Open `index.html` in any modern browser. No build step. No server required.

## Edit it

Open the file in any editor. All CSS lives in the `<style>` block at the top. All JS lives at the bottom. Each section is comment-marked (`<!-- HERO -->`, `<!-- MENU -->`, etc.).

For larger rebuilds, drop `CAVIAR-LOUIE-DESIGN.md` into the Claude Code session and reference Section 13 (prompt template).

## Deploy

Static site. Drag the folder into any of these and it ships:

- Vercel (`vercel deploy`)
- Netlify (drag folder onto netlify.com/drop)
- Cloudflare Pages
- GitHub Pages (enable Pages on this repo, root)

## Brand

Caviar Louie is a Bespoke MKTG client project. Brand guidelines, voice, palette, and component spec live in `CAVIAR-LOUIE-DESIGN.md`. Anything that touches this brand must respect that spec.
